Geodesix language function

Get/Set the current GoogleMap language.

Syntax

Geodesix('language', language)

Returns

String, 2-Letter language code.

Arguments

NameTypeDescription
languageStringSee https://developers.google.com/admin-sdk/directory/v1/languages.

Example

'=Geodesix("language", "") = "EN"

Geodesix mode function

Type of page being displayed in the Map Pane.

Returns

String, Map, Browser or Hidden.

Arguments

NameTypeDescription
modeStringBlank=Just get the mode. Display the map, the browser, or hide the pane.

Geodesix overlay function

Get/Set the visibility of a previously-drawn overlay.

Syntax

Geodesix('overlay', ID, show)

Returns

Boolean, True or False.

Arguments

NameTypeDescription
IDIntegerID number of overlay. Must be 0 or more.
showBooleanBlank=Just get visibility. True=Show it. False=Hide it. Must be one of "", "True", "False".

Example

'=Geodesix("overlay", "12") = True

Geodesix position function

Get/Set the docking position of the map pane.

Syntax

Geodesix('position', quadrant)

Returns

String, Left, Right, Center, Bottom.

Arguments

NameTypeDescription
quadrantStringQuadrant in which to display. Blank=just get quadrant. Must be one of "Centre", "Top", "Bottom", "Left", "Right".

Example

=Geodesix("position") = "Right"

Geodesix setting function

Get/Set the value of a setting.

Syntax

Geodesix('setting', name, value)

Returns

String, Current setting.

Arguments

NameTypeDescription
nameStringName of preference. Must be one of Settings.Names.
valueStringNew value to set. Blank=just get current value.

Example

'=Geodesix("preference", "StartLat") = 42
