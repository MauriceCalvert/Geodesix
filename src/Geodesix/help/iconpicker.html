Geodesix Help - Icon Picker

Good old-fashioned icons

Insert Icon prompts to choose an icon and place its URL in the current cell:

The controls at the top are used to filter and search for icons, which will display in the window.

Double-Clicking an icon inserts the URL, for example http://maps.google.com/mapfiles/kml/pushpin/red-pushpin.png

The modern way

A more extensive source of icons, with over 3'275 cool glyphs, are available from Google Material Symbols:

To use them, type the name of the desired icon in the icon cell (D2 in the example above), replacing spaces with underscores: The icon "Check Circle" is written as "Check_Circle".

The default Material Style is Outlined. Rounded and Sharp can be selected with Tools⇒Drawing, along with the icon size and colour.
