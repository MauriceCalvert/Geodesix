Geodesix Welcome

Thank you for your interest in Geodesix, which is now ready for use. Here are a few tips to assist you:

The tab displayed on the ribbon above is the interface to Geodesix functions, described here. The right-hand button displays the help index, where you will find sample workbooks for common tasks.

Geodesix dialogs have a question-mark button next to the close icon clicking it provides context-sensitive help.

Take a moment to read the getting started tutorial, which will show you how to efficiently create maps.
